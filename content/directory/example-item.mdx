# Example Directory Item

This is an example directory item that demonstrates the wiki-link functionality.

## Wiki-Links

You can link to other items using wiki-link syntax:

- Link with exact slug: [[another-item]]
- Link with display text: [[another-item|Another Item]]

## Features

- **Frontmatter validation** with Zod
- **Wiki-link processing** with fuzzy matching
- **Graph visualization** showing connections
- **Backlinks** showing incoming references

## Content

This item is part of the directory system that supports:

1. Custom fields per directory type
2. Tag-based filtering
3. Full-text search with Pagefind
4. Related items based on tags and links

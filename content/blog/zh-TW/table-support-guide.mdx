## 表格功能

我們的增強表格包含這些強大的功能：

- **🎨 響應式設計**：表格在較小螢幕上會水平滾動
- **🌙 深色模式支援**：自動適應明亮/深色主題
- **✨ 滑鼠懸停效果**：滑鼠懸停時列會高亮以提升使用者體驗
- **🎯 樣式化標題**：使用背景色清楚區分標題
- **📱 行動裝置友善**：針對所有裝置大小最佳化
- **🔧 可自訂**：支援不同的對齊方式和樣式

## 如何使用表格

### 標準 Markdown 表格

只需使用標準的 Markdown 表格語法：

```markdown
| 欄位 1 | 欄位 2 | 欄位 3 |
| -------- | -------- | -------- |
| 資料 1   | 資料 2   | 資料 3   |
```

### 欄位對齊

- **左對齊**：`|-----------|`（預設）
- **置中對齊**：`|:---------:|`
- **右對齊**：`|----------:|`

### 所有對齊方式的範例：

```markdown
| 左對齊 | 置中 | 右對齊 |
| ---- | :----: | ----: |
| 文字 |  文字  |  $100 |
```

## 技術實作

表格樣式由以下技術提供支援：

- **Chakra UI v3**：現代的 React 元件庫
- **remark-gfm**：GitHub 風格的 Markdown 支援
- **響應式設計**：自動行動裝置最佳化
- **主題整合**：無縫的明亮/深色模式切換
- **無障礙功能**：內建螢幕閱讀器支援

您部落格文章中的所有表格都將自動繼承這些美觀的樣式！

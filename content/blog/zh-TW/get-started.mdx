這是一個由 Next.js、Chakra UI、TypeScript 和 MDX 驅動的簡單部落格文章模板。您可以使用此模板創建自己的部落格文章。

由 [sozonome](https://github.com/sozonome) 的 [nextarter-chakra](https://github.com/sozonome/nextarter-chakra) 提供支援，這個模板變得更加簡單。

# 功能與特性

1. 🤖 搜尋引擎優化 (SEO) 友好：[開箱即用的 Next.js 13 SEO](https://nextjs.org/blog/next-13-2#built-in-seo-support-with-new-metadata-api)
2. 🔍 搜尋功能
3. 📚 MDX 支援

- ✅ [基本 Markdown 到 Chakra UI 元件](/blog/basic-markdown-components)
- 🕰️ [自訂元件到 Chakra UI 元件](/blog/custom-components)
- 🕰️ HTML 到 Chakra UI 元件

4. 🌈 使用 Chakra UI 進行樣式設計
5. 🌄 透過 Next.js Open Graph 圖像生成器自動生成封面照片和 OG 圖像

## 先決條件

1. 已安裝 [Node.js](https://nodejs.org/en/) 或 nvm
2. 已安裝 `pnpm`

# 如何在本地環境中使用此模板

1. [克隆此存儲庫](https://github.com/new?template_name=nextjs-chakra-starter-blog&template_owner=kingchun1991)
2. cp .env.local.example .env
3. pnpm Install
4. pnpm dev

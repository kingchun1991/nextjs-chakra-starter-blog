## How to Use Tables

### Standard Markdown Tables

Simply use standard markdown table syntax:

```markdown
| Column 1 | Column 2 | Column 3 |
| -------- | -------- | -------- |
| Data 1   | Data 2   | Data 3   |
```

### Column Alignment

- **Left align**: `|-----------|` (default)
- **Center align**: `|:---------:|`
- **Right align**: `|----------:|`

### Example with all alignments:

```markdown
| Left | Center | Right |
| ---- | :----: | ----: |
| Text |  Text  |  $100 |
```

## Technical Implementation

The table styling is powered by:

- **Chakra UI v3**: Modern React component library
- **remark-gfm**: GitHub Flavored Markdown support
- **Responsive Design**: Automatic mobile optimization
- **Theme Integration**: Seamless light/dark mode switching
- **Accessibility**: Built-in screen reader support

All tables in your blog posts will automatically inherit these beautiful styles!
